# PvP Arena Advanced - Расширенная система мини-игр аватарии

### 🔥 Элементы (Стихии)
6 элементов с уникальными бонусами:

| Элемент | Атака | Защита | HP | Особый эффект | стихии как в рпг-подобных играх
|---------|-------|--------|----|---------------|
| **Огонь** | +15% | -5% | 0% | Burn (горение) |
| **Вода** | -5% | +10% | +20% | Healing (лечение) |
| **Лёд** | 0% | +20% | 0% | Freeze (заморозка) |
| **Молния** | +25% | -10% | -5% | Stun (оглушение) |
| **Земля** | -10% | +25% | +30% | Regeneration |
| **Воздух** | +5% | -15% | 0% | Speed (скорость) |

### Реакции 

Когда атакующщий элемент сталкивается с элементом цели:

- **Огонь + Вода** = Пар (урон -50%)
- **Огонь + Лёд** = Таяние (урон +50%)
- **Огонь + Земля** = Лава (урон +20% + Burn)
- **Вода + Молния** = Электрошок (урон +50% + Stun)
- **Вода + Лёд** = Заморозка (урон 100% + Frozen)
- **Лёд + Молния** = Осколки (урон +30%)
- **Молния + Земля** = Перегрузка (урон +20%)
- **Молния + Воздух** = Буря (урон +40%)
- **Воздух + Огонь** = Инферно (урон +30% + Burn)

## 💡 Стратегии

### Комбинации элементов
1. **Вода + Молния** = Мощный электрошок
2. **Огонь + Воздух** = Инферно (огромный урон)
3. **Лёд + Земля** = Непробиваемая защита

### Тактикп командного боя
1. **Танк (Земля)** + **ДД (Молния)** + **Хил (Вода)**
2. **Берсерк (Огонь)** + **Контроль (Лёд)** + **Саппорт (Воздух)**

### Контер пики типа
- Против Огня → Вода или Лёд
- Против Молнии → Земля
- Против Земли → Молния или Воздух

## Особенности

### Уникальные механики
1. **Элементальный рок-бумага-ножницы** - преимущества/недостатки
2. **Накопление эффектов** - можно иметь несколько баффов
3. **Командные синергии** - бонусы за комбинации элементов
4. **Визуальная обратная связь** - сразу видно что произошло

### Продвинутые функции
- Автовосстановление кулдаунов
- Тики эффектов каждый ход
- Щиты поглощают урон
- Критические удары (шанс 10%)
- Уворот (от элемента Воздух)

## Пример боя

```
Игрок A (Огонь) атакует Игрока B (Вода)

→ Эффект: Burn на 5 тиков (10% HP в тик)

Игрок B использует Щит

Игрок A использует Ультимейт
→ Урон: 100 * 1.8 = 180
→ Игнорирует щит
```
